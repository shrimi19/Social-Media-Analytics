# Social-Media-Analytics
Here for this course we had primarily two projects:
1) Web Graph Network Analysis of IIT Kanpur Wikipedia website : Constructed a large-scale hyperlink network using Screaming Frog and analyzed it in Gephi using PageRank, modularity clustering, and degree metrics to study structural relationships and key nodes within the web graph.
2) Recipe Text Classification Using Machine Learning : Built and evaluated text classification models (Naïve Bayes, Decision Trees, Random Forest) on recipe data using preprocessing, Bag-of-Words (Boolean & TF-IDF), and 10-fold cross-validation to compare performance across accuracy, precision, recall, F1-score, and AUC metrics.
